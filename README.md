Minha atuação foi responsável por **criar toda a arquitetura de dados**, automatizar processos críticos e fornecer análises estratégicas que passaram a sustentar decisões executivas.

## 🚀 **1. Arquitetura de Dados e Governança (Bronze → Silver → Gold)**

### **Entregas Principais**
- Construção de pipelines completos em **Python**, **PostgreSQL**, **Power Query** e **Microsoft Fabric Lakehouse**.
- Implementação das camadas **Bronze, Silver e Gold**, com padronização, tipagem, normalização e modelagem analítica.
- Organização e execução em **Docker**, garantindo reprodutibilidade e escalabilidade.
- Definição de padrões de qualidade, integridade e rastreabilidade.

### **Impacto**
- Dados confiáveis e auditáveis pela primeira vez.
- Redução de divergências entre Sienge, ICX, Anapro e demais sistemas.
- Base sólida para FP&A, vendas, jurídico e controladoria.

<br><br>

## 🔄 **2. Integrações e Automação Corporativa**

### 📌 **Projeto: Sienge API Automation**
- Integração modular com múltiplos endpoints.  
- Execução containerizada (Docker).  
- Scripts organizados por domínio (clientes, unidades, contratos, contas).  
- Logs, autenticação e utilitários padronizados.  
- Automação de rotinas financeiras críticas.

### 📌 **Projeto: parquetData Pipeline**
- Extração de PostgreSQL → Parquet → ADLS → Fabric Lakehouse.  
- Transformações **Bronze → Silver → Gold**.  
- Execução via CLI, Docker e CI/CD (GitHub Actions).  
- Versionamento automático via tags.
- Suporte a pipelines financeiros, contábeis e operacionais.

### 🎯 **Impacto Geral**
- Eliminação de processos manuais.  
- Atualização diária e confiável de dados críticos.  
- Rastreabilidade e governança corporativa.
